Django ORM 기본

WHERE 문 ORM

상품 중에 특정 가격에 대해 조건을 걸고 필터링하고 싶을 때


BETWEEN

상품 중에 특정 가격에 대해 범위를 정해서 필터링하고 싶을 때


LIKE

특정 상품 이름을 찾고 싶을 때


IN

여러 개의 정확한 row를 가져오고 싶을 때
ex) 상품 id가 1, 2인 것을 찾고 싶을 때


AND, OR, NOT

판매하고 있는 상품이면서 동시에 가격이 5000원 이상인 경우 AND

판매하고 있는 상품 또는 가격이 5000원 이상인 경우 OR

판매하지 않는 상품을 제외하고 보고 싶을 경우 NOT


NULL

상품의 알레르기 정보의 기본 값을 NULL로 주었을 때,
알레르기 정보가 있는 것만 보고 싶을 경우

알레르기 정보가 없는 것만 보고 싶을 경우
