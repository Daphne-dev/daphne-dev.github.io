  • create()
    SQLл¬ём—җм„ң INSERTлҘј мқҙмҡ©н•ҙ Tableм—җ лҚ°мқҙн„°лҘј м¶”к°Җн•ҳлҠ” кІғкіј лҸҷмқјн•ҳлӢӨ.

мғқм„ұлҗң мқёмҠӨн„ҙмҠӨлҘј л°ҳнҷҳн•ҙмӨҖлӢӨ.

  • get()
    SQLл¬ём—җм„ң SELECTлҘј мқҙмҡ©н•ҙ Tableмқҳ лҚ°мқҙн„°лҘј мЎ°нҡҢн•ҳлҠ” кІғкіј лҸҷмқјн•ҳлӢӨ.

м§Җм •лҗң мЎ°нҡҢ л§Өк°ң ліҖмҲҳмҷҖ мқјм№ҳн•ҳлҠ” мқёмҠӨн„ҙмҠӨлҘј л°ҳнҷҳ н•ҙмЈјлҠ”лҚ° лӢЁ н•ҳлӮҳмқҳ к°’ л°ҳнҷҳн•ҙмЈјкё° л•Ңл¬ём—җ н•ҙлӢ№ мЎ°кұҙм—җ лҢҖн•ҙ мӨ‘ліөлҗҳлҠ” rowл“Өмқҙ мһҲмңјл©ҙ м—җлҹ¬лҘј мқјмңјнӮЁлӢӨ.

  • filter()
    SQLл¬ём—җм„ң SELECTмҷҖ WHEREм Ҳмқ„ мқҙмҡ©н•ҙ Tableмқҳ лҚ°мқҙн„°лҘј мЎ°нҡҢн•ҳлҠ” кІғкіј лҸҷмқјн•ҳлӢӨ.

мЎ°кұҙм—җ л§һлҠ” лҚ°мқҙн„°л“Өмқ„ QuerySet м•Ҳм—җ л„Јм–ҙм„ң л°ҳнҷҳн•ҙмӨҖлӢӨ.

  • update()
    SQLл¬ём—җм„ң UPDATEлҘј мқҙмҡ©н•ҙ Tableмқҳ лҚ°мқҙн„°лҘј мҲҳм •н•ҳлҠ” кІғкіј лҸҷмқјн•ҳлӢӨ.

мҲҳм •лҗң row к°ңмҲҳлҘј л°ҳнҷҳн•ңлӢӨ.

  • delete()
    SQLл¬ём—җм„ң DELETEлҘј мқҙмҡ©н•ҙ Tableмқҳ лҚ°мқҙн„°лҘј мӮӯм ңн•ҳлҠ” кІғкіј лҸҷмқјн•ҳлӢӨ.

мӮӯм ң мҝјлҰ¬лҘј мҲҳн–ү нӣ„ мӮӯм ңлҗң к°ңмІҙмқҳ мҲҳмҷҖ к°ңмІҙ мң нҳ•лі„ мӮӯм ң нҡҹмҲҳк°Җ мһҲлҠ” dictionaryлҘј л°ҳнҷҳн•ңлӢӨ.

  • save()
    SQLл¬ём—җм„ң INSERT лҳҗлҠ” UPDATEлҘј мҲҳн–үн•ҳлҠ” кІғкіј лҸҷмқјн•ҳлӢӨ.
    лӢЁмқј к°қмІҙм—җ лҢҖн•ҙм„ң м—…лҚ°мқҙнҠёлҘј мҲҳн–үн•  л•Ң л§Һмқҙ мӮ¬мҡ©лҗңлӢӨ.
  • exists()
    filter()мҷҖ н•Ёк»ҳ мӮ¬мҡ©н•ҙм„ң filter мЎ°кұҙм—җ л§һлҠ” лҚ°мқҙн„°к°Җ мһҲлҠ”м§Җ нҷ•мқён•ҳм—¬ мЎҙмһ¬н•ҳл©ҙ True мЎҙмһ¬н•ҳм§Җ м•Ҡмңјл©ҙ FalseлҘј л°ҳнҷҳн•ңлӢӨ.
  • all()
    н•ң н…Ңмқҙлё”мқҳ лӘЁл“  л ҲмҪ”л“ңлҘј к°Җм ёмҳ¬ л•Ң м“°мқёлӢӨ.
  • values()
    лӘЁлҚё мқёмҠӨн„ҙмҠӨк°Җ м•„лӢҢ dictionaryмқ„ нҸ¬н•Ён•ҳлҠ” QuerySetмқ„ л°ҳнҷҳн•ңлӢӨ.
  • values_list()
    values()мҷҖ лӢ¬лҰ¬ нҠңн”Ңмқ„ л°ҳнҷҳн•ңлӢӨ.

Djangoмқҳ кө¬мЎ°

djangoмқҳ кө¬мЎ°лҘј нҒ¬кІҢ ліҙл©ҙ 3к°ңлЎң лӮҳлүҳм–ҙ м§„лӢӨ.
Model, View, TemplateмқҙлӢӨ.

м°ёкі  : django лһҖ?

djangoмқҳ кө¬мЎ°лҘј лҸ„мӢқнҷ”н•ҳл©ҙ м•„лһҳмҷҖ к°ҷмқҙ лҗңлӢӨ.

  • URLs : мҡ”мІӯл°ӣмқҖ URLмқ„ кё°мӨҖмңјлЎң HTTP мҡ”мІӯмқ„ м Ғм Ҳн•ң ViewлЎң ліҙлӮҙмЈјкё° мң„н•ҙ мӮ¬мҡ©лҗңлӢӨ.
  • View : HTTP мҡ”мІӯмқ„ мҲҳмӢ н•ҳкі  HTTP мқ‘лӢөмқ„ л°ҳнҷҳн•ҳлҠ” мҡ”мІӯ мІҳлҰ¬ н•ЁмҲҳмқҙлӢӨ. ViewлҠ” Modelмқ„ нҶөн•ҙ н•„мҡ”н•ң лҚ°мқҙн„°м—җ м ‘к·јн•ҳм—¬ мҡ”мІӯн•ң мӮ¬н•ӯмқ„ мІҳлҰ¬н•ҳкі  Templateм—җ мқ‘лӢөмқҳ м„ңмӢқмқ„ л§ЎкёҙлӢӨ.
  • Model : лҚ°мқҙн„° кө¬мЎ°лҘј м •мқҳн•ҳкі  лҚ°мқҙн„°лІ мқҙмҠӨмқҳ кё°лЎқмқ„ кҙҖлҰ¬н•ҳкі  мҝјлҰ¬н•ҳлҠ” л°©лІ•мқ„ м ңкіөн•ҳлҠ” нҢҢмқҙмҚ¬ к°қмІҙмқҙлӢӨ.
  • ORM : к°қмІҙ(Object)мҷҖ кҙҖкі„нҳ• лҚ°мқҙн„°лІ мқҙмҠӨ(Relational Database)мқҳ лҚ°мқҙн„°лҘј л§Өн•‘(Mapping)н•ҙмЈјлҠ” кІғмқ„ мқҳлҜё, djangoм—җм„ңлҠ” ORMмқ„ нҶөн•ҙ sql мҝјлҰ¬л¬ёмқ„ м“°м§Җ м•Ҡкі лҸ„ лҚ°мқҙн„°лІ мқҙмҠӨмқҳ лҚ°мқҙн„°лҘј лӢӨлЈ° мҲҳ мһҲлӢӨ.
  • Template : нҢҢмқјмқҳ кө¬мЎ°лӮҳ л Ҳмқҙм•„мӣғмқ„ м •мқҳн•ҳкі  Viewм—җм„ң ліҙлӮҙм§„ лҚ°мқҙн„°лҘј л°”нғ•мңјлЎң л ҢлҚ”л§Ғмқ„ н•ҙмЈјлҠ” м—ӯн• мқ„ н•ңлӢӨ.

djangoлҠ” мқјл Ёмқҳ лӢЁкі„лҘј мҲҳн–үн•ҙм„ң мӣ№ лёҢлқјмҡ°м Җм—җ мқ‘лӢөмқ„ л°ҳнҷҳн•ңлӢӨ.

  2. URLsм—җм„ң мҡ”мІӯл°ӣмқҖ URLнҢЁн„ҙм—җ л§һкІҢ м•Ңл§һмқҖ ViewлЎң ліҙлғ„
  3. ViewлҠ” л°ӣмқҖ мҡ”мІӯм—җ л”°лқј Modelм—җм„ң ORMмқ„ мқҙмҡ©н•ҙ DB лҚ°мқҙн„°м—җ м ‘к·јн•ҳм—¬ мҡ”мІӯмқ„ мІҳлҰ¬
  4. HTTP responseлҘј л°ҳнҷҳ

djangoм—җм„ңлҠ” ORMмқ„ нҶөн•ҙ sql мҝјлҰ¬л¬ёмқ„ м“°м§Җ м•Ҡкі лҸ„ лҚ°мқҙн„°лІ мқҙмҠӨмқҳ лҚ°мқҙн„°лҘј лӢӨлЈ° мҲҳ мһҲлӢӨ.


ORMмқҳ мһҘм җ

  • л¶Ҳн•„мҡ”н•ҳкІҢ SQLл¬ёмқ„ кёёкІҢ м Ғм§Җ м•Ҡм•„лҸ„ лҗҳм–ҙ л№ лҘҙкІҢ к°ңл°ңмқҙ к°ҖлҠҘн•ҳлӢӨ.
  • мһҗлҸҷмңјлЎң мІҳлҰ¬н•ҙмЈјкё° л•Ңл¬ём—җ к°ңл°ңмһҗк°Җ мӢ кІҪ м“ё л¶Җл¶„мқҙ мӨ„м–ҙл“ лӢӨ.
  • мң м§ҖліҙмҲҳк°Җ нҺён•ҳкі  мҪ”л“ңмқҳ мһ¬мӮ¬мҡ©мқҙ к°ҖлҠҘн•ҙм§„лӢӨ.

ORMмқҳ лӢЁм җ

  • SQLмқ„ м“°л©ҙ м–ҙл–Ө кіімқҙл“  нҶөмҡ©лҗҳм§Җл§Ң ORMмқҖ н•ҙлӢ№ н”„лЎңк·ёлһҳл°Қ м–ём–ҙл§ҲлӢӨ ORM лқјмқҙлёҢлҹ¬лҰ¬лҘј л”°лЎң л°°мӣҢм•јн•ңлӢӨ.
  • к·ңлӘЁк°Җ нҒ¬кұ°лӮҳ ліөмһЎн•ң н”„лЎңм қнҠёмқҳ кІҪмҡ° SQLлЎң мһ‘м„ұн•ҳлҠ” кІғмқҙ мўӢмқ„ мҲҳлҸ„ мһҲлӢӨ.

нҠ№м • н”„лЎңк·ёлһЁмқ„ к°ңл°ңн•ҳкё° мң„н•ҙ м—¬лҹ¬ мҡ”мҶҢл“Өкіј л©”лүҙм–ј л“ұмқ„ м ңкіөн•ҙмӨҖлӢӨ. лҜёлҰ¬ нӢҖмқҙ м§ңм—¬мһҲкё° л•Ңл¬ём—җ лқјмқҙлёҢлҹ¬лҰ¬м—җ л№„көҗн•ҳл©ҙ мҲҳлҸҷм Ғ, м ңн•ңм Ғмқҙкі  к°Ғк°Ғмқҳ лӘ©м Ғм—җ л§һлҠ” н”„л Ҳмһ„мӣҢнҒ¬лҘј мӮ¬мҡ©н•ҙм•јлҗңлӢӨ.

Library : лӢЁмҲң нҷңмҡ©мқҙ к°ҖлҠҘн•ң лҸ„кө¬л“Өмқҳ м§‘н•©

н•„мҡ”н•  кё°лҠҘмқҙ мһҲмқ„ л•Ңл§ҲлӢӨ н”„л Ҳмһ„мӣҢнҒ¬м—җ л№„н•ҙ лҠҘлҸҷм ҒмңјлЎң к°Җм ёмҷҖм„ң мӮ¬мҡ©н•  мҲҳ мһҲлӢӨ. мӣҗн•ҳлҠ” мҡ©лҸ„м—җ л§һкІҢ н•„мҡ”н•  л•Ң мӮ¬мҡ©н•  мҲҳ мһҲлҠ” лҸ„кө¬мҷҖ к°ҷмқҖ к°ңл…җмқҙлӢӨ.

MVTлһҖ?

MVTлһҖ Djangoм—җм„ң кё°ліём ҒмңјлЎң мӮ¬мҡ©лҗҳлҠ” л””мһҗмқё нҢЁн„ҙмқҙлӢӨ.
мқјл°ҳм Ғмқё Frameworkм—җм„ңлҠ” ліҙнҶө M(model), V(view), C(controller)лЎң мң лӘ…н•ҳлӢӨ.
Djangoм—җм„ңлҠ” Vк°Җ Template, Cк°Җ ViewлЎң м№ҳнҷҳлҗҳм–ҙ мӮ¬мҡ©лҗңлӢӨ.

мҷң көімқҙ мқҙл ҮкІҢ лӮҳлүҳм—Ҳмқ„к№Ң?

мҳҲм „м—җ PHP, ASP, JSPлЎң мӣ№мқ„ к°ңл°ңн•ҳлҚҳ мӢңм Ҳм—җлҠ” лӘЁлҚё, л””мһҗмқё, SQL мІҳлҰ¬ лЎңм§Ғ л“ұмқ„ лӘЁл‘җ н•ң кіім—җ лӘЁм•„м„ң к°ңл°ңмқ„ н–Ҳм—ҲлӢӨ.
н•ҳм§Җл§Ң мқҙлҹ¬н•ң л°©мӢқмқҖ н”„лЎңм қнҠёк°Җ м»Өм§Җл©ҙ м»Өм§ҲмҲҳлЎқ м—¬лҹ¬ л¬ём ңк°Җ л°ңмғқн•ҳмҳҖкі  мқҙлҘј н•ҙкІ°н•ҳкі мһҗ л§Ңл“Өм–ҙмЎҢлӢӨ.
мӣ№ м–ҙн”ҢлҰ¬мјҖмқҙм…ҳмқ„ к°ңл°ңн•ҳлҠ” лҚ°м—җ мһҲм–ҙм„ң мҳҒм—ӯмқ„ нҒ¬кІҢ мң„мҷҖ к°ҷмқҖ 3к°Җм§ҖлЎң лӮҳлҲҲ кІғмқҙлӢӨ.
мқҙлҘј нҶөн•ҙ лҸ…лҰҪм ҒмңјлЎң к°ңл°ңн•  мҲҳ мһҲкІҢ лҗҳм—Ҳкі  нҳ‘м—…м—җ мң лҰ¬н•ҙмЎҢлӢӨ.
