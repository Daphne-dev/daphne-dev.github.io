CSSмқҳ кө¬м„ұ

  • span : selector(м„ нғқмһҗ)
  • color : property
  • red : value

styleмқ„ HTML нҺҳмқҙм§Җм—җ м Ғмҡ©н•ҳлҠ” 3к°Җм§Җ л°©лІ•

1. inline
HTML нғңк·ё м•Ҳм—җлӢӨк°Җ м Ғмҡ©н•ңлӢӨ.
лӢӨлҘё CSSнҢҢмқјм—җ м Ғмҡ©н•ң кІғліҙлӢӨ к°ҖмһҘ лЁјм Җ м Ғмҡ©лҗңлӢӨ.

2. internal
style нғңк·ёлЎң м§Җм •н•ңлӢӨ.
кө¬мЎ°мҷҖ мҠӨнғҖмқјмқҙ м„һмқҙкІҢ лҗҳлҜҖлЎң мң м§ҖліҙмҲҳк°Җ м–ҙл өлӢӨ.
лі„лҸ„мқҳ CSS нҢҢмқјмқ„ кҙҖлҰ¬н•ҳм§Җ м•Ҡм•„лҸ„ лҗҳл©° м„ңлІ„м—җ CSS нҢҢмқјмқ„ л¶ҖлҘҙкё° мң„н•ҙ лі„лҸ„мқҳ лёҢлқјмҡ°м Җк°Җ мҡ”мІӯмқ„ ліҙлӮј н•„мҡ”к°Җ м—ҶлӢӨ.

3. external
мҷёл¶ҖнҢҢмқј(.css)лЎң м§Җм •н•ҳлҠ” л°©мӢқ
мЈјлЎң мқҙ л°©лІ•мқ„ л§Һмқҙ мӮ¬мҡ©н•ңлӢӨ.
css мҪ”л“ңлҘј кө¬нҳ„н•ҳкі , style.cssмҷҖ к°ҷмқҖ лі„лҸ„ нҢҢмқјлЎң л§Ңл“ лӢӨ.
мқҙнӣ„ linkнғңк·ёлЎң м¶”к°Җн•ҳл©ҙ лҗңлӢӨ.

4. мҡ°м„ мҲңмң„
inlineмқҖ лі„лҸ„мқҳ мҡ°м„ мҲңмң„лҘј к°–м§Җл§Ң, internalкіј externalмқҖ мҡ°м„ мҲңмң„к°Җ лҸҷл“ұн•ҳлӢӨ. л”°лқјм„ң кІ№м№ҳлҠ” м„ м–ёмқҙ мһҲмқ„ кІҪмҡ° лӮҳмӨ‘м—җ м„ м–ёлҗң мҶҚм„ұмқҙ л°ҳмҳҒлҗңлӢӨ.

classмҷҖ id мҶҚм„ұ

ID

  • кі мң н•ң мҶҚм„ұмңјлЎң н•ң HTML л¬ём„ңм—җ н•ҳлӮҳл§Ң мӮ¬мҡ©к°ҖлҠҘн•ҳлӢӨ.
  • кі мң н•ң ID к°’мқҙ мһҲмңјл©ҙ н•ҳлӮҳн•ҳлӮҳм—җ нҠ№лі„н•ң м ңм–ҙлҘј н•  мҲҳ мһҲмңјл©° кІҖмғүм—җлҸ„ мҡ©мқҙн•ҳлӢӨ.

Class

  • н•ҳлӮҳмқҳ HTML л¬ём„ң м•Ҳм—җ мӨ‘ліөн•ҙм„ң мӮ¬мҡ©к°ҖлҠҘн•ҳлӢӨ.

  • нҷҲнҺҳмқҙм§Җ м „мІҙм Ғмқё мҠӨнғҖмқјмқ„ мқјкҙҖм„ұ мһҲкІҢ м§Җм •н•ҳкё° мң„н•ҙ class мӮ¬мҡ©мқҖ н•„мҲҳм ҒмқҙлӢӨ.
