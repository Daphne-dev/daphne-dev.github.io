HTTP(Hypertext Transfer Protocol)лһҖ?

  • м„ңлІ„мҷҖ нҒҙлқјмқҙм–ёнҠёк°Җ мқён„°л„·мғҒм—җм„ң лҚ°мқҙн„°лҘј мЈјкі л°ӣкё° мң„н•ң н”„лЎңнҶ мҪң(protocol)

HTTP мһ‘лҸҷл°©мӢқ

  • Server/Client лӘЁлҚёмқ„ л”°лҘёлӢӨ.
  • нҒҙлқјмқҙм–ёнҠём—җм„ң мҡ”мІӯ(request)лҘј ліҙлӮҙл©ҙ м„ңлІ„лҠ” мҡ”мІӯмқ„ мІҳлҰ¬н•ҙм„ң мқ‘лӢө(reponse)н•ҳлҠ” кө¬мЎ°

HTTP (Hypertext Transfer Protocol)

  • мҡ”мІӯ л©”м„ңл“ң : GET, PUT, POST, PUSH, OPTIONS л“ұмқҳ мҡ”мІӯ л°©мӢқмқҙ мҳЁлӢӨ.
  • мҡ”мІӯ URI : мҡ”мІӯн•ҳлҠ” мһҗмӣҗмқҳ мң„м№ҳлҘј лӘ…мӢңн•ңлӢӨ.
  • HTTP н”„лЎңнҶ мҪң лІ„м „ : мӣ№ лёҢлқјмҡ°м Җк°Җ мӮ¬мҡ©н•ҳлҠ” н”„лЎңнҶ мҪң лІ„м „мқҙлӢӨ.

к°Ғк°Ғмқҳ л©”мҶҢл“ңл“Ө

  • GET : м •ліҙлҘј мҡ”мІӯн•ҳкё° мң„н•ҙм„ң мӮ¬мҡ©н•ңлӢӨ. (SELECT)
  • POST : м •ліҙлҘј л°Җм–ҙл„Јкё° мң„н•ҙм„ң мӮ¬мҡ©н•ңлӢӨ. (INSERT)
  • PUT : м •ліҙлҘј м—…лҚ°мқҙнҠён•ҳкё° мң„н•ҙм„ң мӮ¬мҡ©н•ңлӢӨ. (UPDATE)
  • DELETE : м •ліҙлҘј мӮӯм ңн•ҳкё° мң„н•ҙм„ң мӮ¬мҡ©н•ңлӢӨ. (DELETE)
  • HEAD : (HTTP)н—ӨлҚ” м •ліҙл§Ң мҡ”мІӯн•ңлӢӨ. н•ҙлӢ№ мһҗмӣҗмқҙ мЎҙмһ¬н•ҳлҠ”м§Җ нҳ№мқҖ м„ңлІ„м—җ л¬ём ңк°Җ м—ҶлҠ”м§ҖлҘј нҷ•мқён•ҳкё° мң„н•ҙм„ң мӮ¬мҡ©н•ңлӢӨ.
  • OPTIONS : мӣ№м„ңлІ„к°Җ м§Җмӣҗн•ҳлҠ” л©”м„ңл“ңмқҳ мў…лҘҳлҘј мҡ”мІӯн•ңлӢӨ.
  • TRACE : нҒҙлқјмқҙм–ёнҠёмқҳ мҡ”мІӯмқ„ к·ёлҢҖлЎң л°ҳнҷҳн•ңлӢӨ. мҳҲм»ЁлҢҖ echo м„ңл№„мҠӨлЎң м„ңлІ„ мғҒнғңлҘј нҷ•мқён•ҳкё° мң„н•ң лӘ©м ҒмңјлЎң мЈјлЎң мӮ¬мҡ©н•ңлӢӨ.
