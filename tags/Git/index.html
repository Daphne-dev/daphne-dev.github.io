2020-10-02Git9л¶„ мқҪкё° (лҢҖлһө 1375 лӢЁм–ҙ)

Git кё°ліё 1

GitмқҙлһҖ?

git кіөмӢқ нҷҲнҺҳмқҙм§Җм—җм„ң gitм—җ лҢҖн•ҙ м„ӨлӘ…н•ң лӮҙмҡ©мқҙлӢӨ.

Git кіөмӢқ нҷҲнҺҳмқҙм§Җ

Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.

мүҪкІҢ л§җн•ҳл©ҙ н”„лЎңм қнҠё нҢҢмқјмқҳ ліҖкІҪ мӮ¬н•ӯмқ„ м¶”м Ғн•ҳлҠ” мӢңмҠӨн…ңмқёлҚ° мқҙлҹ¬н•ң Gitмқ„ нҶөн•ҙ к°ңл°ңмһҗл“ӨмқҖ н”„лЎңм қнҠёмқҳ ліҖкІҪ мӮ¬н•ӯмқ„ кё°лЎқн•ҳкі , нҠ№м • мӢңм җмқҳ лІ„м „мңјлЎң м–ём ңл“  лҸҢм•„к°Ҳ мҲҳ мһҲкІҢ лҗҳм—ҲлӢӨ.

мқҙлҹ¬н•ң GitмқҖ нҳ‘м—…н•  л•Ң лҚ”мҡұ нҡЁкіјк°Җ к·№лҢҖнҷ” лҗҳлҠ”лҚ° к°Ғк°Ғмқҳ к°ңл°ңмһҗк°Җ к°Ғкё° лӢӨлҘё н”„лЎңм қнҠё лІ„м „мқ„ ліёмқё м»ҙн“Ён„°м—җ к°–кІҢ н•ҳм—¬ к°ңл°ңмқ„ м§„н–үн•ң нӣ„ мқҙлҹ¬н•ң к°ңлі„ лІ„м „мқҳ н”„лЎңм қнҠёл“Өмқ„ Gitмқ„ нҶөн•ҙ лі‘н•©н•ҳм—¬ кё°мӨҖмқҙ лҗҳлҠ” лІ„м „мқҳ н”„лЎңм қнҠём—җ м Ғмҡ©н•ҳлҠ” кІғмқҙ мүҪкІҢ к°ҖлҠҘн•ҙмЎҢлӢӨ.

мҰү, Gitмқ„ мӮ¬мҡ©н•ЁмңјлЎңмҚЁ мІҙкі„м Ғмқё к°ңл°ңмқҙ к°ҖлҠҘн•ҙм§ҖкІҢ лҗҳм—ҲлӢӨ.

Git м„Өм№ҳн•ҳкё°

Git лӢӨмҡҙлЎңл“ң л§ҒнҒ¬

мһҗмӢ мқҙ мӮ¬мҡ©н•ҳлҠ” OSм—җ л§һлҠ” кІғмқ„ м„ нғқн•ҙ м„Өм№ҳн•ҳл©ҙ лҗңлӢӨ.
м„Өм№ҳлҘј л§ҲміӨлӢӨл©ҙ н„°лҜёл„җ нҳ№мқҖ CMDм—җм„ң м•„лһҳмқҳ м»Өл§Ёл“ңлҘј мһ…л Ҙн•ҙліҙмһҗ.

Gitмқҙ м •мғҒм ҒмңјлЎң м„Өм№ҳлҗҳм—ҲлӢӨл©ҙ, м„Өм№ҳлҗҳм–ҙ мһҲлҠ” Git лІ„м „мқҙ н‘ңмӢңлҗңлӢӨ.

Repositories(м ҖмһҘмҶҢ)

Gitм—җм„ңлҠ” repositoryлқјлҠ” мҡ©м–ҙк°Җ мһҲлӢӨ.
Git repositoryлҠ” GitмңјлЎң кҙҖлҰ¬н•ҳлҠ” н”„лЎңм қнҠё м ҖмһҘмҶҢмқҙлӢӨ.
н”„лЎңм қнҠёлҘј м ҖмһҘн•ҳлҠ” н•ҳлӮҳмқҳ нҸҙлҚ”лқјкі  ліҙл©ҙ лҗңлӢӨ.

Git repositoryлҠ” нҒ¬кІҢ л‘җ к°Җм§Җ мў…лҘҳк°Җ мһҲлӢӨ.

  • Local repository : ліёмқёмқҳ м»ҙн“Ён„°м—җ м ҖмһҘлҗң лЎңм»¬ лІ„м „мқҳ н”„лЎңм қнҠё м ҖмһҘмҶҢ
  • Remote repository : мҷёл¶Җ(мӣҗкІ© м„ңлІ„)м—җ мһҲлҠ” мҷёл¶Җ лІ„м „мқҳ н”„лЎңм қнҠё м ҖмһҘмҶҢ

ліҙнҶө Local repositoryм—җ Remote repositoryмқҳ н”„лЎңм қнҠёлҘј к°Җм ёмҷҖм„ң мҲҳм • л°Ҹ к°ңл°ңмқ„ н•ҳкі  Remote repositoryм—җ лЎңм»¬м—җм„ң мһ‘м—…н•ң мӮ¬н•ӯл“Өмқ„ лі‘н•©н•ҳкі  ліҖкІҪ мӮ¬н•ӯмқ„ м Ғмҡ©н•ңлӢӨ.

2. Initilizing a repository

GitмңјлЎң н”„лЎңм қнҠё кҙҖлҰ¬лҘј мӢңмһ‘н•ҳл Өл©ҙ н”„лЎңм қнҠё нҸҙлҚ”лЎң мҷҖм„ң лӢӨмқҢ лӘ…л №м–ҙлҘј мһ…л Ҙн•ҳл©ҙ лҗңлӢӨ.

мқҙ лӘ…л №м–ҙлҠ” н”„лЎңм қнҠё лӮҙм—җ мҲЁкІЁм§„ .git л””л үнҶ лҰ¬лҘј мғқм„ұн•ҳлҠ”лҚ° мқҙм ң GitмқҖ нҳ„мһ¬ м ҖмһҘмҶҢм—җ лҢҖн•ҙ лӘЁл“  ліҖкІҪмӮ¬н•ӯмқ„ м¶”м Ғ/кҙҖлҰ¬н•  мҲҳ мһҲкІҢ лҗңлӢӨ.

3. Staging л°Ҹ мғҒнғң нҷ•мқён•ҳкё°

Gitм—җм„ңлҠ” лӘЁл“  нҢҢмқјмқҙ нҒ¬кІҢ Gitмқҙ м¶”м ҒмӨ‘мқё нҢҢмқјкіј м•„лӢҢ нҢҢмқјлЎң лӮҳлүңлӢӨ.
Gitм—җм„ңлҠ” лӢӨлҘё лІ„м „ кҙҖлҰ¬ лҸ„кө¬мҷҖ лӢӨлҘё Staging AreaлқјлҠ” кіөк°„мқҙ мһҲлӢӨ.

лӘЁл“  нҢҢмқјмқҖ н•ӯмғҒ Staging AreaлқјлҠ” мӨ‘к°„ лӢЁкі„лҘј кұ°міҗм„ң ліҖкІҪмӮ¬н•ӯмқҙ м ҖмһҘмҶҢм—җ м»Өл°Ӣмқҙ лҗңлӢӨ.

мқҙлҘј нҶөн•ҙ м–»лҠ” мң мҡ©н•ң м җмқҙ мһҲлҠ”лҚ°

  • мқјл¶Җл¶„л§Ң м»Өл°Ӣн•  л–„
  • м¶©лҸҢмқ„ н•ҙкІ°н•  л•Ң
  • м»Өл°Ӣмқ„ лӢӨмӢңн•  л•Ң

н”„лЎңм қнҠёлҘј н•ҳлӢӨліҙл©ҙ мқјл¶Җл¶„л§Ң м»Өл°Ӣмқ„ н•ҳкі  мӢ¶мқ„ л•ҢлҸ„ мһҲмқ„ мҲҳ мһҲлӢӨ.
л§Ңм•Ҫ м»Өл°Ӣмқ„ н–ҲлҠ”лҚ° м¶©лҸҢмқҙ лӮ¬мқ„ кІҪмҡ° Staging AreaлҘј нҷ•мқён•ҳл©ҙ м–ҙл–Ө нҢҢмқјмқҙ м¶©лҸҢн–ҲлҠ”м§Җ л°”лЎң м•Ң мҲҳ мһҲлӢӨ.
м»Өл°Ӣмқ„ н–ҲлҠ”лҚ° м»Өл°Ӣн•ң кІғмқҙ л§ҲмқҢм—җ м•Ҳл“Өм–ҙм„ң лӢӨмӢң мҲҳм •н•ҳкі  м»Өл°Ӣн•  л–„лҸ„ мң мҡ©н•ҳлӢӨ.

repositoryмқҳ нҳ„мһ¬ мғҒнғңлҘј нҷ•мқён•ҳл Өл©ҙ лӢӨмқҢ лӘ…л №м–ҙлҘј мһ…л Ҙн•ҳл©ҙ лҗңлӢӨ.

н•ҙлӢ№ лӘ…л №м–ҙлҘј нҶөн•ҙ м–ҙл–Ө нҢҢмқјмқҙ ліҖкІҪлҗҳм—ҲлҠ”м§Җ, м–ҙл–Ө нҢҢмқјмқҙ м¶”к°Җлҗҳм—ҲлҠ”м§ҖлҘј лӘЁл‘җ м•Ң мҲҳ мһҲлӢӨ.
лҳҗн•ң GitмңјлЎң кҙҖлҰ¬лҗҳкі  мһҲлҠ” нҢҢмқјмқём§Җ м•„лӢҢм§ҖлҸ„ м•Ң мҲҳ мһҲлӢӨ.

4. Staging files(Staging areaм—җ нҢҢмқј м¶”к°Җн•ҳкё°)
