нҢҢмқҙмҚ¬мқҖ м•„лһҳмқҳ мҲңм„ңлҢҖлЎң лӘЁл“Ҳ/packageлҘј м°ҫлҠ”лӢӨ.

pythonм—җм„ңлҠ” import н•ҳкі мһҗ н•ҳлҠ” лӘЁл“Ҳкіј packageлҘј м°ҫмқ„ л•Ңм—җ мң„мҷҖ к°ҷмқҖ мҲңм„ңлЎң м°ҫкІҢ лҗҳм§Җл§Ң л§Ңмқј sys.pathм—җм„ңлҸ„ лӘ»м°ҫмқ„ кІҪмҡ°

м•„лһҳмҷҖ к°ҷмқҙ м§Ғм ‘ нҷ•мқён•ҙліј мҲҳ мһҲлӢӨ.


sys.modules vs sys.path

sys.modulesлҠ” dictionary мһҗлЈҢнҳ•мңјлЎң module/package л°Ҹ к°Ғк°Ғмқҳ кІҪлЎңлҘј м ҖмһҘн•ҳкі  мһҲкі 
sys.pathлҠ” list мһҗлЈҢнҳ•мңјлЎң module/packageк°Җ м„Өм№ҳлҗң dictionary кІҪлЎңлҘј лӢҙкі  мһҲлӢӨ.

sysлҘј л¶Ҳлҹ¬мҳ¬ мҲҳ мһҲлҠ” мқҙмң 

sys.modulesлӮҳ sys.pathлҠ” sys лӘЁл“ҲлЎңл¶Җн„° мҳӨлҠ” кІғмқ„ м•Ң мҲҳ мһҲлӢӨ.
к·ёл ҮлӢӨл©ҙ sys лӘЁл“ҲмқҖ м–ҙл–»кІҢ м°ҫм•„м„ң л¶Ҳлҹ¬мҳ¬ мҲҳ мһҲмқ„к№Ң?

sys.modulesлҘј м¶ңл Ҙн•ҙліҙл©ҙ dictionary нҳ•нғңлЎң sysк°Җ м ҖмһҘлҗң кІғмқ„ ліј мҲҳ мһҲлӢӨ.

мң„м—җм„ң м•Ң мҲҳ мһҲл“Ҝмқҙ built-in modulesм—җм„ң л¶Ҳлҹ¬мҳӨкё° л•Ңл¬ём—җ л”°лЎң кІҪлЎңлҘј м„Өм •н•ҳм§Җ м•Ҡм•„лҸ„ sysлҘј л¶Ҳлҹ¬мҳ¬ мҲҳ мһҲлҠ” кІғмқ„ м•Ң мҲҳ мһҲлӢӨ.

Absolute pathмҷҖ Relative pathмқҳ м°Ёмқҙм җ

pythonм—җм„ң лҜёлҰ¬ м ңкіөлҗҳлҠ” moduleмқҙлӮҳ packageк°Җ м•„лӢҢ м§Ғм ‘ л§Ңл“  local packageлҠ” м–ҙл–»кІҢ л¶Ҳлҹ¬мҳ¬ мҲҳ мһҲмқ„к№Ң?


мқҙлҘј мң„н•ҙ м ҲлҢҖ кІҪлЎң(Absolute path)мҷҖ мғҒлҢҖ кІҪлЎң(Relative path)м—җ лҢҖн•ҙ м•Ңм•„м•ј н•ңлӢӨ.
м ҲлҢҖ кІҪлЎңлҠ” л§җ к·ёлҢҖлЎң м–ҙл””м—җм„ң л¶Ҳлҹ¬мҳӨл“  н•ҙлӢ№ packageмқҳ мң„м№ҳм—җ лҢҖн•ҙ н•ӯмғҒ лҸҷмқјн•ң кІҪлЎңлҘј к°Җм§Җкі  мһҲкё° л•Ңл¬ём—җ л¶Ҳлҹ¬мҳ¬ мҲҳ мһҲлӢӨ.

лӢӨмқҢкіј к°ҷмқҖ н”„лЎңм қнҠёлҘј мҳҲлЎң л“Өм–ҙліҙмһҗ.

my_appмқҙлқјлҠ” н”„лЎңм қнҠёмқҙл©° package1, package2лқјлҠ” 2к°ңмқҳ packageлҘј к°Җм§Җкі  мһҲлӢӨ.

м ҲлҢҖ кІҪлЎң(Absolute path)лҘј мқҙмҡ©н•ҙ package1кіј package2лҘј importн•ҳл©ҙ лӢӨмқҢкіј к°ҷлӢӨ.

my_app н”„лЎңм қнҠёмқҳ мөңмғҒмң„ л””л үнҶ лҰ¬м—җм„ң л¶Җн„° мӢңмһ‘н•ҳлҠ” кІғмқ„ м•Ң мҲҳ мһҲлҠ”лҚ°, мқҙмІҳлҹј my_app н”„лЎңм қнҠё лӮҙм—җм„ң м–ҙлҠҗ нҢҢмқј, м–ҙлҠҗ мң„м№ҳм—җм„ң import н•ҳлҚҳм§Җ кІҪлЎңк°Җ н•ӯмғҒ мң„мҷҖ к°ҷмқҙ лҸҷмқјн•ҳкІҢ лҗҳлҠ” кІғмқ„ Absolute pathлқјкі  н•ңлӢӨ.

мғҒлҢҖ кІҪлЎң(Relative path)лҠ” нҳ„мһ¬ importлҘј н•ҳлҠ” мң„м№ҳм—җ л”°лқј кІҪлЎңк°Җ лӢ¬лқјм§ҖлҠ” кІғмқ„ л§җн•ңлӢӨ.

м•„лһҳмқҳ мҳҲмӢңлҘј ліҙмһҗ

мң„м—җм„ң dot(.)мқҖ importк°Җ м„ м–ёлҗҳлҠ” нҢҢмқјмқҳ нҳ„мһ¬ мң„м№ҳлҘј л§җн•ңлӢӨ.
dot 2к°ң(..)лҠ” нҳ„мһ¬ мң„м№ҳм—җм„ң н•ң лӢЁкі„ мғҒмң„ л””л үнҶ лҰ¬лЎң к°ҖлҠ” кІғмқ„ мқҳлҜён•ңлӢӨ.

мқҙмҷҖ к°ҷмқҙ import н•ҳлҠ” мң„м№ҳлҘј кё°мӨҖмңјлЎң кІҪлЎңлҘј м •мқҳн•ҳлҠ” кІғмқ„ Relative pathлқјкі  н•ңлӢӨ.

к·ёл ҮлӢӨл©ҙ м§Ғм ‘ к°ңл°ңн•ң local packageлҘј import н•  л•ҢлҘј м•Ңм•„ліҙмһҗ.

мқјл°ҳм ҒмңјлЎң Absolute pathлҘј мӮ¬мҡ©н•ңлӢӨ.
лӢӨл§Ң Absolute pathлҘј мӮ¬мҡ©н•ҳл©ҙ н•ң к°Җм§Җ лӢЁм җмқҙ мһҲлҠ”лҚ° кІҪлЎңк°Җ л„Ҳл¬ҙ кёём–ҙм§„лӢӨлҠ” м җмқҙлӢӨ.
мқҙлҘј ліҙмҷ„н•ҳкё°мң„н•ҙ м Ғм Ҳн•ҳкІҢ Relative pathлҘј мӮ¬мҡ©н•ҙмӨҖлӢӨ.


calculator нҢЁнӮӨм§Җ л§Ңл“Өм–ҙліҙкё°

мң„мқҳ мҪ”л“ңлҘј ліҙл©ҙ main.pyм—җм„ң add_and_multiply.pyлҘј import н–Ҳмқ„ л•Ң м—җлҹ¬к°Җ л°ңмғқн•ңлӢӨ.
м•„лһҳмҷҖ к°ҷмқҖ м—җлҹ¬ л©”мӢңм§Җк°Җ лң¬лӢӨ.

нҢҢмқҙмҚ¬м—җм„ңлҠ” к°Ғк°Ғмқҳ нҢҢмқјл§ҲлӢӨ мқҙлҰ„("__name__")мқ„ лҜёлҰ¬ л°ӣм•„лҶ“м•„ м ҖмһҘн•ҳлҠ” кіімқҙ мһҲлӢӨ. мқҙлҘј кё°мӨҖмңјлЎң нҢҢмқҙмҚ¬ мқён„°н”„лҰ¬н„°лҠ” к°Ғк°Ғмқҳ нҢҢмқјм—җ лҢҖн•ң кІҪлЎңлҘј м•Ң мҲҳ мһҲкІҢ лҗҳлҠ”лҚ° к·ё мӨ‘м—җм„ң "__main__"мқҖ мЎ°кёҲ нҠ№лі„н•ҳлӢӨ.

н•ӯмғҒ мӢңмһ‘м җмқҙ "__main__"мңјлЎң лҗҳкё° л•Ңл¬ём—җ мӢңмһ‘м җм—җ л”°лқј "__main__"мқҙ л°”лҖҢлҠ”лҚ° мқҙлҹ° мқҙмң лЎң нҢҢмқҙмҚ¬мқҳ мӢңмһ‘м җмқҙ лҗҳлҠ” нҢҢмқҙмҚ¬ м–ҙн”ҢлҰ¬мјҖмқҙм…ҳмқҖ н•ӯмғҒ м ҲлҢҖ кІҪлЎңлҘј мӮ¬мҡ©н•ҙм•јн•ңлӢӨ.

    м ңл„Ҳл Ҳмқҙн„° н•ЁмҲҳмқё yield мҷём—җлҸ„ м ңл„Ҳл Ҳмқҙн„° н‘ңнҳ„мӢқ(generator expression)мқҙ мһҲлӢӨ.

    list comprehension vs lazy evaluation

    list comprehensionкіј lazy evaluationмқ„ л№„көҗн•ҙліҙмһҗ

    list comprehensionмқҖ мқҙн„°л Ҳмқҙн„° мҡ”мҶҢк°Җ лӘЁл‘җ мғқм„ұлҗ  л•Ңк№Ңм§Җ кё°лӢӨлҰ¬лӢӨк°Җ м „лӢ¬н•ҙмЈјлҠ” л°ҳл©ҙ
    lazy evaluationмқҖ мқҙн„°л Ҳмқҙн„° мҡ”мҶҢлҘј мғқм„ұн•  л•Ңл§ҲлӢӨ мҰүмӢң м „лӢ¬н•ҙмЈјлҠ” кІғмқ„ ліј мҲҳ мһҲлӢӨ.
