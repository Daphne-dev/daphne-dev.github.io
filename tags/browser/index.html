browser의 동작

  1. HTML을 해석해서 DOM Tree를 생성, CSS를 해석해서 CSS Tree(CSS Object Model)을 만든다.
  2. DOM Tree와 CSS Tree는 Render Tree로 다시 조합된다.
  3. Render Tree정보를 통해서 화면에 어떻게 나타낼지 Painting과정을 거치게 된다.
