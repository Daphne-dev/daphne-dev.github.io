[JavaScript] Callback 함수

Callback function이란?

함수로써 다른 함수에 인수로 전달되어 외부 함수 내에서 호출되는 것

여기서 외부 함수란 Callback function 밖의 함수들을 말한다.
다음 예시를 보자.

[출력결과]

printNumber 함수를 호출하는데, 이 떄 callBack 함수를 인자로 주었다.
printNumber 함수가 실행되고 그 안에서 callBack 함수를 호출하여 callBack 함수를 실행한다.
